第二课 git的其余命令行操作

一、复习
git init             初始化仓库。新建仓库
git clone 线上仓库地址 文件夹名    克隆仓库     
git status           查看状态
git add 文件/ .       添加至暂存区
git commit -m "注释"    提交暂存区至本地仓库
git log   (oneline)(all)   查看版本日志
git checkout ##            跳转    
git push -u ## ##          提交至远程仓库

二、标签
git tag             查看所有标签
git tag -a -m ###   添加标签
git show ##         查看标签内容
git checkout ##     快速跳跃

三、分支
git branch ##           创建分支
git checkout ##         切换到分支
git log --all --graph   图形化显示
git checkout -b ##      创建分支并跳转
git merge ##            合并分支

四、远程仓库的进一步讲解
git remote add ## ##
git push -u ## ##
git pull
